## How to contribute
Always make your contributions for the latest `develop` branch, not `master`, so it can be tracked for the next release.

## Development
*Requires: NodeJS and Yarn*

### Step 1: Install packages.
```
yarn install
```
### Step 2: Start NodeJS server.
```
yarn start
```
